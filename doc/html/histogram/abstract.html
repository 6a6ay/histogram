Abstract

This C++11 library implements two easy-to-use and powerful n-dimensional histogram classes, using a policy-based design, optimized for convenience, customizability, and highest performance.

Two histogram implementations in C++ are included. static_histogram exploits compile-time information as much as possible to provide the maximum performance, but has no run-time flexibility. dynamic_histogram makes the opposite trade-off. The two classes share an interface. Optional Python bindings for the dynamic_histogram are included, based on Boost.Python. Optional serialization support is implemented, based on Boost.Serialization.

The histograms have value semantics. Move operations and trips over the language boundary from C++ to Python and back are cheap. Histograms can be streamed from/to files and pickled in Python. Numpy is supported to speed up operations in Python: histograms can be filled with Numpy arrays at high speed (faster than Numpy's own histogram functions) and are convertible into Numpy arrays without copying data.

My goal is to submit this project to Boost, that's why it uses the Boost directory structure and namespace. The code is released under the Boost Software License.
